AI REFLECTION

The game:
I have made a tower defense game, with modern ui.
You have multiple towers that do different things and cost different amounts of money.
You have a boss wave, that has more hp than normal enemies.
You can chose different maps and difficulties.


The reflection:

The start of the project was going great, no errors and it everything worked without having to ask to fix stuff.
Then i decided i want to add more to the project and it went downhill, the ai was having troubles with making good code that fitted with the other scripts.
When i wanted to revamp the whole css and html file the ai couldnt send it to me because there was a limit.
I did like that the ai could make a whole game with one prompt, and i could play the tower defense game.
I also really liked that the ai was giving me ideas for the project so i could do less.


Limitation:

To my next point the limit on the ai was so bad that i hit it 20 minutes into the lesson.
It was different based on how big the question was for the ai, but most of time i hit it in 1 hour.
I also hated how the ai made me search a function in the 1000+ line script, because he couldnt update the whole script because the limit of the free model.
But the most irritating part of this project was just doing nothing and waiting on the code.


I learned a few things about ai this lesson:

Ai is very good in making a project based on how the ai wants it, BUT it is a different story when ai needs to make changes that i want.
I have also noticed that ai can make a project, but the code is just garbage unreadable mess. If i needed to work on this project without ai, i wouldnt be able to do it.
I like the use of ai to make a function, or algorithm but i want to understand it before i use it. 
When i started coding i just copied code of ai, and then tried to make it work (didin't work and this was before i went to school), and it felt defeating. 
So when i make a project now, maybe i ask for help to make a function but i would never copy paste it without understanding it. 


End of the reflection:

Ai has made a functional game that u can play, but it wasnt as easy as i had expected. 
I dont hate the use of ai, but i try not to copy and paste code that ai makes.
I do like that ai can explain code and helps me understand it and i also get better at coding because of it.
