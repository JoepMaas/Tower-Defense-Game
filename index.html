HOW TO PLAY

Single Player: Quick Race — drive solo on any map.
Multiplayer: Click Multiplayer Lobby → Invite/Join → Create or Join a room using the 6-digit code.
Controls:
W / ↑ — Accelerate
S / ↓ — Brake / Reverse
A D — Steer
SPACE — Handbrake
ESC — Pause / Return

Network: Uses WebRTC peer-to-peer. The host shares their room code. Up to 4 players. Each client connects directly — no server needed beyond signaling.
